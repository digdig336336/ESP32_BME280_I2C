# ESP32_BME280_I2C
BME280 library for I2C interface ( for Arduino - ESP32 ).  
Beta ver1.1  
  
これはBOSCH製　温度・湿度・気圧センサー 用のI2C通信ライブラリです。  
Arduino core for ESP32 専用です。
